# Atchaya Jyothi Panchang - ஷார்ட்கோடு வழிகாட்டி

## பிளகின் ஷார்ட்கோடு (Plugin Shortcode)

இந்த பிளகின்னை உங்கள் WordPress பக்கங்கள் அல்லது இடுகைகளில் பயன்படுத்த பின்வரும் ஷார்ட்கோடை சேர்க்கவும்:

```
[atchaya_panchang]
```

## ஷார்ட்கோடு விருப்பங்கள் (Shortcode Options)

| விருப்பம் | விளக்கம் | இயல்புநிலை | எடுத்துக்காட்டு |
|---------|---------|-----------|-------------|
| `place` | பஞ்சாங்கம் கணக்கிடப்பட வேண்டிய இடத்தின் பெயர் | Vellore | `place="Chennai"` |
| `date` | பஞ்சாங்கம் கணக்கிடப்பட வேண்டிய தேதி (YYYY-MM-DD) | இன்றைய தேதி | `date="2023-11-15"` |
| `lang` | மொழி தேர்வு (ta/en/te/ka/hi/ml) | WordPress தளத்தின் மொழி | `lang="en"` |
| `style` | ராசி கட்டத்தின் பாணி (south/north/east) | south | `style="south"` |
| `show_gowri` | கௌரி பஞ்சாங்கத்தைக் காண்பிக்க (yes/no) | yes | `show_gowri="yes"` |
| `show_panchapakshi` | பஞ்ச பட்சியைக் காண்பிக்க (yes/no) | yes | `show_panchapakshi="yes"` |

## ஆதரவு மொழிகள் (Supported Languages)

- **ta** - தமிழ் (Tamil)
- **en** - English (ஆங்கிலம்)
- **te** - తెలుగు (Telugu)
- **ka** - ಕನ್ನಡ (Kannada)
- **hi** - हिन्दी (Hindi)
- **ml** - മലയാളം (Malayalam)

## எடுத்துக்காட்டுகள் (Examples)

### 1. இயல்புநிலை பஞ்சாங்கம்
```
[atchaya_panchang]
```
இன்றைய தேதிக்கு Vellore இடத்திற்கு WordPress தளத்தின் மொழியில் பஞ்சாங்கத்தைக் காண்பிக்கும்.

### 2. தமிழ் மொழியில் சென்னைக்கு பஞ்சாங்கம்
```
[atchaya_panchang place="Chennai" lang="ta"]
```

### 3. ஆங்கிலத்தில் மதுரைக்கு குறிப்பிட்ட தேதிக்கு பஞ்சாங்கம்
```
[atchaya_panchang place="Madurai" date="2023-11-15" lang="en"]
```

### 4. தெలుగு மொழியில் வெள்ளூருக்கு பஞ்சாங்கம்
```
[atchaya_panchang place="Vellore" lang="te"]
```

### 5. கன்னட மொழியில் பெங்களூருக்கு பஞ்சாங்கம்
```
[atchaya_panchang place="Bangalore" lang="ka"]
```

### 6. ஹிந்தியில் தில்லிக்கு பஞ்சாங்கம்
```
[atchaya_panchang place="Delhi" lang="hi"]
```

### 7. മലയാളത്തിൽ കൊച്ചിക്കു പഞ്ചാംഗം
```
[atchaya_panchang place="Kochi" lang="ml"]
```

## பிளகின் அம்சங்கள் (Plugin Features)

✅ **தினசரி தானியங்கி மாற்றம்** - பஞ்சாங்க விவரங்கள் தினசரி தானாகவே மாறும்.

✅ **ஊர் தேடல்** - OpenStreetMap Nominatim API பயன்படுத்தி ஊர் தேடல் வசதி.

✅ **மொழி தேர்வு** - 6 மொழிகளுக்கு ஆதரவு (தமிழ், ஆங்கிலம், తెలుగు, ಕನ್ನಡ, हिन्दी, മലയാളം).

✅ **தேதி தேர்வு** - எந்த தேதிக்கும் பஞ்சாங்கம் பெற முடியும்.

✅ **அழகான வடிவமைப்பு** - நீலம் மற்றும் தங்க நிற பாரம்பரிய தோற்றம்.

✅ **AJAX ভিত্তিক** - பக்கத்தை மீண்டும் ஏற்றாமல் பஞ்சாங்கம் புதுப்பிக்கப்படும்.

## நிறுவல் (Installation)

1. இந்த பிளகின் ফோல்டரை `/wp-content/plugins/` இல் அப்ளோட் செய்யவும்.
2. WordPress நிர்வாகத்தில் "Plugins" பக்கத்திற்குச் செல்லவும்.
3. "Atchaya Jyothi Panchang" பிளகின்னை செயல்படுத்தவும்.
4. எந்த WordPress பக்கம் அல்லது இடுகையிலும் `[atchaya_panchang]` ஷார்ட்கோடைப் பயன்படுத்தவும்.
