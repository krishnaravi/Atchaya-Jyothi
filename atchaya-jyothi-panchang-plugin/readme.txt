Full Dynamic Tamil Panchangam with Swiss Ephemeris / Prokerala API support.

== Description ==

Atchaya Jyothi Panchang provides a comprehensive daily panchang for Tamil-speaking users. It includes Tithi, Nakshatra, Yoga, Karana, Gowri, Rahu Kalam, and a South Indian Rasi Chart.

== Installation ==

1. Upload the `atchaya-jyothi-panchang` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[atchaya_panchang]` in any page or post.

== Shortcode Options ==

- `place`: Set the location (default: Vellore)
- `date`: Set the date (default: today)
- `style`: 'south' or 'north' for the chart style.

Example: `[atchaya_panchang place="Chennai" style="south"]`
